# TFG - Scripts de Matlab
Carpeta 1 "Modulación AM" contiene:
- transmision_seno: fichero con pruebas de transmisión de un seno mediante la Pluto empleando un cable coaxial (Funcional)
- simulacion_AM_exp_compleja: fichero con una simulación de un transmisión AM usando como portadora una exponencial compleja. Incluye el receptor y el procesado adecuados. (Funcional)
- transmision_AM_exp_compleja: fichero con una transmisión AM usando como portadora una exponencial compleja mediante la Pluto con un cable coaxial. Incluye el receptor y el procesado adecuados. (No Funcional, Pendiente de revisión).
